	1. System Events: C++ Core, libuv
	2. Custom Events: from Javascript Core -> It's Event Emitter inside Javscript core

#Event Emitter:

#Call and Apply:
    - function.call() & function.apply(): cùng ý nghĩa
    - nếu function có tham số: function (param1, param2...)
        -> function.call(thisArg, param1, param2) (cách nhau bởi dẩy phẩy)
        -> function.apply (thisArg [param1, param2])...-> gọi theo mảng
    - Ở hàm function cứ chỗ nào có this thì thay bằng thisArg()
    - Dùng nhiều trong "borrowing method/function"

#Clip 9 - khi dùng hàm Constructor.call(this) -> sẽ truyền các attribute của constructor vào this
    -> cách này đảm bảo truyền mọi attributes của Constructor vào this
